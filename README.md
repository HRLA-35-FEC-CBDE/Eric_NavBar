## Usage

- Install dependencies.
  - [ ] npm install
- Run scripts.
  - [ ] npm start
  - [ ] npm run dev:react
  - [ ] npm run seed (Only run if you want to populate the database with items)
- Start mongod in terminal or command prompt (If necessary)
